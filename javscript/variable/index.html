Javascripts has 3 primitives data types

  • Numbers, eg. 123, 3.56 etc
  • Strings of text e.g. 'This is text value'
  • Boolean e.g true or false

Two trivial data types

the two trivial data types are null and undefined, each of which defines a single value

Object

JavaScript also has composite data type known as object

All numbers are represented as 64-bit floating point values
